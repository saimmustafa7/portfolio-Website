# Saim Mustafa — Personal Portfolio

A minimal, responsive personal portfolio website built with pure HTML and CSS.

---

## 📁 Project Structure

```
portfolio/
├── index.html
├── styles/
│   └── style.css
└── images/
    ├── logo.png
    ├── me.png
    ├── ClashDisplay-Semibold.otf
    ├── 1.jpg
    ├── 2.jpg
    └── 3.jpg
```

---

## 🚀 Features

- **Smooth scrolling** via [Lenis](https://github.com/darkroomengineering/lenis)
- **Sticky "Projects" heading** with parallax-style scroll effect
- **Hamburger nav menu** for tablet and mobile
- **Fully responsive** — desktop, tablet (≤1024px), and mobile (≤430px)
- **Custom font** — Clash Display Semibold (local `.otf`)

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling & layout |
| Lenis (CDN) | Smooth scroll |
| Clash Display | Custom heading font |

No frameworks. No build tools. Just open `index.html` in a browser.

---

## 📱 Responsive Breakpoints

| Breakpoint | Target |
|---|---|
| `≤ 1024px` | Tablets — stacked layout, hamburger menu |
| `≤ 768px` | Large phones — scaled fonts, stacked footer |
| `≤ 430px` | Mobile phones — compact layout, full-width sections |

---

## 🔗 Sections

- **Hero** — Full-viewport background image with name heading
- **About** — Short bio with stat highlights (hours, curiosity, passion)
- **Areas of Interest** — Animated table of CS topics
- **Work** — Project description and scrollable project images
- **Footer** — Links to Instagram, LinkedIn, and Email
